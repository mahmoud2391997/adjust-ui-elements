# Tinytales E-commerce

A modern e-commerce product details page built with Next.js and Tailwind CSS.

## Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Open browser:**
   Navigate to `http://localhost:3000`

## Features

- Responsive product details page
- Product image carousel
- Rating and reviews system
- Similar products section
- Authentication redirect system

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Poppins & Inter fonts
